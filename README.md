# is-not-ten-thousand 

> Return true if the given number is even.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save is-not-ten-thousand
```

## Usage

```js
var isNotTenThousand = require('{%= name %}');

isNotTenThousand(0);
//=> true
```
